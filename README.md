# iOS Club — VIT Bhopal 

## Project Description
This is the official landing page for the **iOS Club at VIT Bhopal University**. The iOS Club is a premier student-run development community dedicated to the Apple ecosystem. The website serves as a central hub for students to discover club activities, explore educational programs (like Swift, SwiftUI, and CoreML), check upcoming events, meet the core team, and apply for the upcoming cohorts. 

The website features a modern, premium "Glassmorphism" aesthetic with smooth scroll animations, animated background orbs, and a fully responsive layout.

## Features
* **Modern UI/UX:** Premium dark-theme design utilizing Glassmorphism (frosted glass effects) and vibrant gradient accents.
* **Smooth Animations:** Custom CSS keyframe animations for floating elements, background orbs, and gradient borders.
* **Scroll Reveals:** Content elegantly fades and slides into view as the user scrolls, powered by the `IntersectionObserver` API.
* **Responsive Design:** Flawlessly adapts to all screen sizes, featuring a custom hamburger menu for mobile devices.
* **Dynamic Sections:**
    * **Programs:** Highlights core learning tracks (Swift Foundations, SwiftUI, AI & CoreML, App Launch Sprint).
    * **Events Calendar:** Showcases upcoming hackathons, workshops, and WWDC watch parties.
    * **Team Showcase:** Profiles the core student leadership team.
* **Interactive CTA:** A functional application form interface with basic front-end email validation.

## Tech Stack Used
This project is built using pure, vanilla web technologies to ensure maximum performance and zero dependencies.
* **HTML5:** Semantic markup and structuring.
* **CSS3:** * CSS Variables for consistent theming.
    * CSS Grid & Flexbox for layout management.
    * Backdrop-filters for the glass effect.
* **Vanilla JavaScript:** * `IntersectionObserver` for scroll-triggered animations.
    * Event listeners for mobile menu toggling and form validation.

## Setup Instructions
Since this project uses no external frameworks, bundlers, or package managers, running it locally is incredibly simple.

### Prerequisites
* A modern web browser (Safari, Chrome, Firefox, Edge).
* *(Optional)* A code editor like Visual Studio Code.

### Running Locally
1. **Download the files:** Clone this repository or download the source code as a ZIP file.
2. **Extract the files** (if downloaded as a ZIP).
3. **Open the site:** * **Method 1:** Simply double-click the `index.html` file to open it directly in your default web browser.
   * **Method 2 (Recommended):** If you are using VS Code, install the **Live Server** extension. Right-click the `index.html` file and select "Open with Live Server" to view the site with automatic hot-reloading as you make changes.
